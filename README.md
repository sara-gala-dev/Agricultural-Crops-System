# Agricultural-Crops-System
##About the System  
A desktop application built with C#-NET to manage all operations of an agricultural crops company that imports potatoes from farmers and exports to factories and traders

##Business Overview
-Buy potatoes from farmers (with advance payments and seeds)
-Store in refrigerated warehouses
-Sell to traders and factories (e.g. chips factories) 
-Sell seeds to farmers 

##Database Tables
-Persons:  Farmers, customers, companies, suppliers.
-potato _Types:  Types and seasons of potatoes. 
-Store_place: Refrigerated warehouses with responsible employee daily_stores seed - Daily stock movement.  
-FarmarAccount details: Farmer accounts with discounts.
-CustomerAccount details:  Trader accounts. 
-CampanyAccount_details: Factory accounts.
-Employees & Salaries: Monthly, weekly, daily.
-Income & Outcome : financial transactions.
-Cash: Daily financial summary.

##Tecnologies
-C#.NET
-Microsoft SQL Server
-Stored Procedures
-ADO.NET
